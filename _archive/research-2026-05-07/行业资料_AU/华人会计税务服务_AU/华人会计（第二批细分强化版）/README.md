# 华人会计（第二批细分强化版）资料包

用途：补充第一版《华人会计 / 税务服务资料包》，把更容易转化的细分场景、本地 suburb 语境、FAQ 型内容和广告表达补深，方便后续直接生成页面、短文案、广告素材和销售沟通脚本。

当前目录文件：
- `00-行业研究.md`
- `01-本地样本.md`
- `02-内容表达与广告话术.md`
- `03-五文件映射建议.md`

本补充包重点：
- 个人报税：工资族、个体户、投资收入、海外收入、ATO 补报/更正
- 小企业会计：Pty Ltd、sole trader、partnership、family trust 的常见场景
- BAS / bookkeeping / payroll：季度申报、Xero/MYOB、STP、super、雇主合规
- 投资房 / CGT / trust FAQ：出租物业、负扣税、折旧、出售、family trust 分配与边界
- suburb 化表达：Hurstville、Chatswood、Sydney CBD，并兼顾 Burwood / Parramatta 可延展语境

说明：
- 语境仍以澳大利亚 / 悉尼本地华人市场为主。
- 本资料用于品牌、页面、社媒、投放与销售前置研究，不构成税务、法律或财务意见。
- 涉及 trust、CGT、税务居民、海外收入等复杂情形时，后续落地文案应保留“按个人情况评估”的合规边界。
