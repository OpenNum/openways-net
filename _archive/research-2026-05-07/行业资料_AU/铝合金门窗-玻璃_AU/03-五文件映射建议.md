# 五文件映射建议

结论：**可以映射到 5 文件，而且非常适合。**

原因：
1. 这个行业天然存在“高频维修 / 单项定制”和“整套 replacement / 商业项目”两层结构。
2. 搜索意图很清楚，适合按项目拆独立落地页。
3. 用户决策特别依赖具体场景，例如 shower screen、sliding door repair、security doors、shopfront glass，不适合只做一个大而泛的总页。

## 推荐映射方案 A（最通用）
1. **首页 / 总入口**
   - 目标：概览品牌、服务范围、住宅/商业覆盖、流程、核心卖点、CTA
   - 关键词：aluminium windows and doors sydney / glass services sydney

2. **Shower Screens 页面**
   - 目标：承接浴室翻新和局部升级高频需求
   - 内容：frameless、semi-frameless、fully framed、mirrors、custom layouts、measure & install

3. **Sliding Doors & Window Repairs 页面**
   - 目标：承接 repair 型强需求
   - 内容：rollers、tracks、locks、glass replacement、window adjustments、repair vs replace

4. **Security Doors & Flyscreens 页面**
   - 目标：承接澳洲本地非常真实的通风、防虫、安全需求
   - 内容：security screen doors、window screens、flyscreens、custom fit、ventilation + protection

5. **Aluminium Windows / Shopfront / Commercial Glazing 页面**
   - 目标：承接更高客单或商业询盘
   - 内容：window replacement、sliding / stacking doors、shopfront framing、office glazing、supply & install

## 可选映射方案 B（更偏商业化）
把第 3 页和第 5 页稍微重构为：
- **Glass Replacement & Repairs 页面**
- **Shopfront & Commercial Glazing 页面**

适用情况：
- 团队维修响应快，想吃更多 emergency / quick quote 需求
- 同时希望把 commercial fitout 和 retail frontage 拉开表达

## 每页核心意图建议

### 1. 首页
解决“你们到底做哪些门窗玻璃服务、住宅和商业都做吗、怎么开始咨询”。

### 2. Shower Screens 页
解决“我现在就想做淋浴房，要选哪种，能不能定制，装出来好不好看”。

### 3. Repairs 页
解决“门窗现在不好用，先修还是换，多久能看，价格逻辑是什么”。

### 4. Security / Flyscreen 页
解决“我想通风、防虫、更安全，你们能提供什么方案”。

### 5. Aluminium / Commercial 页
解决“我要整套门窗升级，或者我有店面 / office / shopfront 项目，你们能不能统筹”。

## 页面结构提醒
每页都建议包含：
- Sydney / service area 场景标题
- 适合谁
- 做哪些类型
- repair / replace 或 supply / install 的边界
- 常见痛点
- 流程说明
- trust signals
- FAQ
- CTA

## 品牌方向建议

### 如果走大众住宅路线
关键词重点放：
- practical
- reliable
- made to measure
- smooth operation
- tidy installation
- clear quote

### 如果走华人路线
可强化：
- 中文沟通
- 发照片先沟通
- 先判断修还是换
- shower screen / sliding door / flyscreen / security door 一起处理
- 尺寸、玻璃、五金和预算先说清楚

### 如果走中高端升级路线
关键词重点放：
- custom
- clean lines
- premium hardware
- energy-efficient glazing
- modern finish
- project-managed installation

### 如果走小商业 / B2B 路线
关键词重点放：
- shopfront
- commercial glazing
- office partition
- fitout support
- fast turnaround
- professional frontage

## 最后建议
第一版资料和页面，建议先不要把所有玻璃类业务铺得太散。最稳的一组是：
- shower screens
- sliding door / window repairs
- security doors & flyscreens
- aluminium windows & doors
- shopfront / commercial glazing

这样既有高频住宅转化入口，也有更高客单和商业项目的升级路径。
