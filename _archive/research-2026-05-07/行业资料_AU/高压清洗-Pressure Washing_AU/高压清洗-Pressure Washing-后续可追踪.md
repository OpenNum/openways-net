# 高压清洗-Pressure Washing-后续可追踪

## 1. 建议持续跟踪的指标
- 线索来源：Google / Facebook / Instagram / 小红书 / 转介绍 / 地图搜索
- 询盘类型：driveway / roof / exterior / commercial
- 报价方式：照片初报 / 上门复核
- 成交率：住宅 vs 商业
- 附加销售率：driveway 升级 bundle、roof + gutter、exterior + driveway
- 完工后评价率
- 90 天复购率

## 2. 建议补抓的竞品信息
- 澳洲本地按城市收集：Sydney / Melbourne / Brisbane / Perth / Adelaide
- 每城至少 10 家
- 字段建议：
  - 是否主打 driveway
  - 是否单列 roof cleaning
  - 是否强调 soft wash
  - 是否展示 before-after
  - 是否写 insured / guarantee
  - 是否支持 online quote / free quote
  - 是否主打商业维护

## 3. 建议验证的业务假设
1. Driveway 是否为最佳广告引流入口
2. Roof 是否更适合高客单但必须现场复核
3. Exterior bundle 是否最容易提升客单价
4. 商业客户是否更适合季度维护合约
5. “发照片先报价”是否显著提高私信转化率

## 4. 后续可补充的数据文件
- 城市竞品表.csv
- 报价规则版本记录.md
- 常见材质风险库.md
- 施工前后素材命名规范.md
- 商业维护提案模板.md

## 5. 适合后续新增的 AI 自动化
- 从 WhatsApp / 表单自动读取图片并初分诊
- 根据 suburb 和项目类型自动路由到不同报价模板
- 自动生成报价摘要 PDF
- 完工后自动生成案例卡片
- 自动提醒客户复洗周期

- 澳洲各州关于高处作业、药剂排放、公共区域清洗的合规要求
- 不同屋顶材质对应的清洗方式边界
