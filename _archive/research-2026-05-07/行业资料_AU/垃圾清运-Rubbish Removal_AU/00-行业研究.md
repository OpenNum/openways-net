# 00-行业研究

## 1. 行业定义
垃圾清运 / Junk Removal 属于“上门搬运 + 运输 + 分类处置”型本地服务，典型服务包括家庭杂物清理、搬家前后清空、家具家电清运、办公室清场、装修废料、花园垃圾、遗产房与囤积清理。

它和 skip bin / dumpster rental 的核心差异是：
- 客户不自己搬，服务商负责人工搬运
- 更强调“快”，大量需求来自临时、急单、验房、退租、搬家、装修节点
- 报价常以体积、装车占比、单件、难度、楼层、距离、禁运风险综合形成
- 获客强依赖本地搜索、Google Maps、落地页、电话、短信、照片沟通

## 2. 典型商业模式
### 2.1 定价方式
1. 体积计价
- 以立方米、立方码、卡车 1/8、1/4、1/2、整车计价
- 澳洲本地样本常出现“每立方米起价”
- 美国大品牌常用 truck volume increments

2. 单件计价
- 适合床垫、沙发、冰箱、电视、洗衣机、轮胎等标准件
- 适合线上快速报价和广告投放

3. 到场 fixed quote
- 先给范围价或照片预估，到场后给最终一口价
- 对楼层、拆解、重物、混装废弃物更稳妥

4. 附加费
- 上下楼、远距离搬运、拆解、超重、特殊回收、停车困难、等待时间、夜间/急单

### 2.2 服务承诺
- Same day / within hours
- 免费到场估价或免费快速预估
- 全包价，含人工、装车、清扫、处置费
- 不满意可不做
- 回收 / 捐赠优先，减少 landfill

## 3. 客户需求触发点
- 房东 / 中介验房前
- 退租、搬家、搬办公室
- 家具家电换新
- 物业 / strata / 商铺清场
- 装修后废料堆积
- 亲属离世后的遗产房清理
- 囤积、地下室、车库、后院积压
- 市政 bulky pickup 太慢或规则太严格

## 4. 行业关键流程
### 4.1 前台链路
投放/搜索 -> 电话/表单/短信/WhatsApp -> 询问地址、照片、品类、体积、时效 -> 预估 -> 预约时间窗 -> 到场确认 -> 装运 -> 付款 -> 回访/评价

### 4.2 后台链路
线索录入 -> 分诊 -> 禁运判断 -> 报价 -> 排线 -> 派车 -> ETA 通知 -> 完工照片 -> 处置记录 -> 评价与复购

## 5. 高价值运营点
### 5.1 同日服务
同日服务是强转化抓手，特别适用于：
- 退租检查
- 紧急清场
- 家具送新前旧件移除
- 中介、物业、办公室临时清空

行业常见表达：
- same day service
- arrive within hours
- booked in the morning, picked up today
- fast, fixed-price quote

同日服务的本质不是口号，而是调度能力：
- 需要可插单时段
- 需要地理聚类与线路弹性
- 需要客服在首轮就判断“今天能不能接、是否值得插”

### 5.2 照片报价
照片报价非常适合这个行业，因为客户常难以准确描述体积。
推荐收集：
- 全景照 + 近景照
- 单件尺寸参照物
- 楼层、电梯、台阶、停车点
- 是否需要拆解
- 是否混有禁运物

照片报价最好区分三层：
1. AI 远程预估价
2. 人工复核价
3. 到场最终确认价

### 5.3 体积计价
体积计价是规模化的核心，因为：
- 易标准化
- 易培训客服和司机
- 易和卡车装载率绑定
- 易和广告页面可视化表达联动

推荐统一单位：
- 对外可展示：立方米 / 卡车占比
- 对内换算：标准装载格、司机经验系数、重量与处置风险系数

### 5.4 禁运物管理
大多数全服务清运公司都强调“几乎什么都收，但 hazardous / 非法 / 高风险除外”。
常见禁运或需特殊处理：
- 化学品、溶剂、油漆、油品
- 石棉 / asbestos
- 油桶、储油罐
- 易燃易爆物
- 液体废弃物
- 某些医疗废弃物
- 某些工业危险品

所以客服分诊必须把“物品能否收”放在前面，而不是等到司机上门才发现。

## 6. 客服分诊框架
首轮至少拿到 8 类信息：
1. 地址与 suburb / zip
2. 希望时间，今天/明天/本周
3. 垃圾类型
4. 大件数量
5. 体积范围
6. 位置与搬运难度，室内/室外/楼层/电梯/台阶
7. 是否有照片
8. 是否含禁运物

建议把线索分成 5 档：
- A：标准小单，可快速照片报价
- B：标准中单，可给范围价并预约
- C：大单或整屋，需人工复核
- D：疑似禁运，先合规确认
- E：超出范围，不接或转介

## 7. 路线与调度
垃圾清运比普通配送更复杂，因为每单耗时不确定，装车量持续变化。

调度核心变量：
- 地理位置
- 预估装车体积
- 预估作业时长
- 上下楼/拆解难度
- 车辆剩余容量
- tipping run 是否需要插入
- 同日 SLA

建议调度规则：
1. 上午保留同日急单 buffer
2. 按区域聚类排线
3. 司机端实时回传剩余容量
4. 高不确定单放在线路后段或独立车组
5. 大件/重件单提前标记工具需求
6. 让客服承诺时间窗，不轻易承诺精确分钟

## 8. AI 最适合接入的流程
### 8.1 线索接待 AI
- 自动回复表单、短信、WhatsApp
- 引导用户发照片、地址、时间要求
- 自动判别是否同日可接

### 8.2 报价 AI
- 识别照片中的物品类别和大致体积
- 生成价格区间和风险标签
- 标记“需到场确认”的原因

### 8.3 合规 AI
- 拦截禁运物
- 输出标准解释话术
- 建议替代处置渠道

### 8.4 调度 AI
- 根据 suburb、容量、ETA、作业时长推荐插单
- 生成司机路线优先级
- 自动给客户发送 ETA 更新

### 8.5 复购 AI
- 完工后索评
- 针对搬家、物业、中介、装修客做再营销
- 沉淀 B2B 常客，如物业经理、房产中介、装修队

## 9. 对 AI 产品化最重要的行业结论
1. 这是一个非常适合“照片 + 表单 + 调度”的行业。
2. 用户并不想研究规则，只想快速知道“能不能收、多少钱、什么时候来”。
3. 报价不是一次性动作，而是“预估 -> 复核 -> 到场确认”的漏斗。
4. 同日服务不是广告词，而是排线、容量、短信沟通、司机回传共同作用的结果。
5. 禁运物拦截越前置，越省掉无效出车与差评。
6. 体积计价必须配合可视化示例，否则用户很难自报准确。
7. 适合把客服 SOP 结构化，做成 AI 可执行规则库。
