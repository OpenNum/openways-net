# 上门维修-Mobile Mechanic - 五文件映射建议

更新时间：2026-04-12

## 建议对应的 AI 行动包 5 文件

1. Quick Start Guide
- 哪类问题适合上门维修
- 最容易成交的入口
- 典型客户场景

2. Tool Guide
- 地址 / 车型 / 症状收集
- FAQ 与客服辅助
- 预约说明模板
- 价格说明模板

3. Prompt Library
- 服务介绍提示词
- 故障初判提示词
- 上门预约提示词
- 能修 / 需送厂解释提示词

4. Workflow SOP
- 询盘到预约
- 地址 / 车型 / 问题确认
- 上门检查或服务
- 结果说明
- 售后与转化跟进

5. 7-Day Plan
- 第一天梳理服务入口
- 第二天整理 FAQ
- 第三天搭建咨询表单
- 第四天写页面样本
- 第五天补故障脚本
- 第六天补预约和价格说明
- 第七天收口标准流程
