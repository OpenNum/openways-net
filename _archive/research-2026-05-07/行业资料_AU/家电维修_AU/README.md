# 家电维修行业资料

本目录用于给“家电维修”赛道做快速进入、内容策划、广告转化与 AI 行动包映射。

## 文件结构
- `00-行业研究.md`：行业概况、需求类型、用户决策、竞争格局、风险点
- `01-本地样本.md`：澳洲本地样本、平台样本、可直接参考的话术与服务结构
- `02-内容表达与广告话术.md`：适合短视频、落地页、广告、客服转化的话术模板
- `03-五文件映射建议.md`：按 AI 行动包格式输出，覆盖 Quick Start Guide、Tool Guide、Prompt Library、Workflow SOP、7-Day Plan
- `家电维修-后续可追踪.md`：后续继续补齐的数据点、样本池、验证动作
- `sources.md`：本轮公开来源记录

## 本轮结论摘要
1. 家电维修是明显的“高意图、强本地、强时效、强信任”服务。
2. 用户最关心的不是“修不修”，而是“多久到、先收多少、值不值得修、有没有保修”。
3. 本地商家普遍强调：same day、qualified technician、warranty、all brands、residential/commercial。
4. 内容和广告转化重点应放在：故障场景、时效承诺、透明报价、保修、品牌覆盖、上门便利。
5. 若做 AI 行动包，最适合切入的是：线索收集、报价预判、客服分诊、内容批量生成、SOP 标准化。
