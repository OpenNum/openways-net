说明：
- 语境以当地 / 您所在的城市本地服务为主。
- 本包把“杀虫”和“除草”放在同一资料包里处理，但实际业务上通常分成两条主线：
  1. **Pest Control**：住宅 / 商业病虫害防治，重点是蟑螂、白蚁、老鼠、蜘蛛、蚂蚁、床虱、黄蜂、possums 等。
  2. **Weed Control / Weed Spraying**：草坪、花园、空地、property management body、商业场地的杂草治理，常与园艺 / lawn care / grounds maintenance 重叠。
- 后续做站时建议先明确主入口是“住宅杀虫灭鼠”还是“草坪除草喷药”，再决定单品牌还是双服务线并列。 
- 本次重点覆盖：行业细分、目标客户、服务项目、获客渠道、痛点、卖点、本地样本、常见中英表达，以及是否适合映射到后续 5 文件。
