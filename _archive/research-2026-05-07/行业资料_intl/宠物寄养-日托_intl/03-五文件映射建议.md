# 宠物寄养-日托 — 五文件映射建议

## 文件01 Quick Start Guide
- 核心角度：寄养/日托老板最先可用 AI 做 FAQ、首次咨询回复、规则说明和假期文案
- 支撑来源：00-行业研究中的信任门槛与规则密度；01-本地样本中的 assessment / surcharge / updates 等真实场景；02-内容表达与广告话术中的常见沟通话术
- 重点痛点：客户不够放心；规则解释容易引发摩擦

## 文件02 Tool Guide
- 核心角度：适合客服起稿、规则文档、英文改写、内容规划和图片/视频配文工具
- 重点工具类型：内容、客服、翻译改写、排版、图文辅助
- 行业特殊需求：涉及药物、健康、攻击性、行为判断时，AI 只能辅助整理表达，不能替代专业评估

## 文件03 Prompt Library
- 核心角度：老板最需要 AI 帮写的是首次咨询、评估说明、取消政策、每日更新、旺季提醒、价格解释
- 建议分组：
  - 一、首次咨询与试托教育
  - 二、寄养/日托规则说明
  - 三、每日更新与状态反馈
  - 四、旺季营销与名额提醒
  - 五、特殊情况与投诉处理
- 关键话术来源：02 文件中的首次咨询、入住前准备、价格说明、取消政策、接送沟通；01 文件中的本地门店卖点

## 文件04 Workflow SOP
- 核心角度：首次咨询、评估安排、入住交接、在托更新、离店回访这几个流程最值得 AI 介入
- 典型工作流：客户咨询 → 评估/试托 → 确认名额与付款 → 入住/入托 → 日常更新 → 接回与复购
- AI 接入点：
  - FAQ 和首次回复
  - 规则、条款、准备清单说明
  - 每日状态更新与回访文案

## 文件05 7-Day Plan
- 核心角度：先用 AI 解决规则和 FAQ，再做内容营销和复购跟进
- 难点预判：很多老板知道规则，但不会写成让客户容易接受的文字
- 第一天建议：先把最常见的 5 个客户问题交给 AI，生成可直接复制使用的回复模板
