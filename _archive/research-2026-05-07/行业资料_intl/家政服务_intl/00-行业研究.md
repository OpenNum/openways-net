# 家政服务行业研究（当地 / 您所在的城市）

更新时间：2026-04-09

## 一句话结论
您所在的城市语境下，“家政服务”最稳定、最容易标准化获客的不是传统住家保姆叙事，而是以 **house cleaning / regular cleaning / deep cleaning / end of lease cleaning / domestic assistance** 为核心的家庭服务组合。市场表达强调：**省时间、可信任、可在线预约、价格清晰、同一清洁员、保险齐全、可退租验房、适合忙碌家庭与租房人群**。如果往服务延展，则 NDIS、aged care、yard care、整理收纳、地毯/窗户/烤箱等附加项目很常见。

## 1. 行业细分

### A. 常规家庭清洁
最典型：weekly / fortnightly / monthly。

常见场景：
- 双职工家庭
- 有孩子家庭
- 公寓住户
- 忙碌白领
- 不想把周末花在打扫上的人

### B. 深度清洁 / spring clean
适用于：
- 长期未彻底清洁
- 节日前后
- 客人来访前
- 霉斑、水垢、厨房油污明显

### C. 退租清洁 / bond clean / end of lease clean
您所在的城市非常强需求的单项。

典型特征：
- 与租房搬家强绑定
- 用户对“bond back / inspection-ready”极敏感
- 常需 inside oven / inside windows / wall spot clean / carpet steam clean 等细项

### D. 搬入 / 搬出清洁
和 end of lease 相邻，但话术偏“入住前卫生重置”。

### E. 家务协助 / housekeeping / domestic assistance
比纯 cleaning 更接近日常家政，可能包含：
- tidying / folding / linen change
- dishwashing
- laundry
- pantry organising
- bed making
- 基础整理

### F. NDIS / disability cleaning
当地本地一个很重要的细分。

特点：
- 强调 participant、support coordinator、plan manager
- 要讲 service agreement、plan managed / self managed / agency managed
- 重点卖点不是“便宜”，而是“稳定、尊重、按计划执行、沟通顺畅”

### G. aged care / 老年人居家协助
常与 domestic assistance、轻家务、陪伴式支持叠加。

### H. 附加专项服务
常见作为 upsell：
- oven cleaning
- rangehood cleaning
- fridge cleaning
- interior windows
- carpet steam cleaning
- upholstery
- balcony / alfresco
- yard care / gardening
- pressure cleaning

## 2. 目标客户

### 主要客户画像
1. **忙碌双职工家庭**
   - 核心诉求：省时间、稳定、省心
2. **有孩子 / 有宠物家庭**
   - 核心诉求：卫生、安全、固定阿姨/清洁员
3. **租房搬家人群**
   - 核心诉求：顺利退租、押金尽量退回、清单覆盖完整
4. **高压白领 / 高收入公寓住户**
   - 核心诉求：快速预约、准时、细节到位、无需反复沟通
5. **NDIS 参与者及其协调人 / 家属**
   - 核心诉求：按 funding 走、服务稳定、可持续配合
6. **老年人家庭 / 子女代下单**
   - 核心诉求：可信任、礼貌、持续上门、轻体力家务支持
7. **房东 / 物业经理 / Airbnb 房源持有者**
   - 核心诉求：周转效率、检查标准、稳定交付

## 5. 用户痛点

### 购买前痛点
- 不知道谁靠谱，怕踩雷
- 怕陌生人进家里
- 怕报价不透明，现场加价
- 不知道哪些内容是否包含
- 搬家时间紧，怕约不到

### 交付中痛点
- 清洁员迟到 / 放鸽子
- 每次来的人不同，要重复交代
- 只做表面，不做细节
- 忽略厨房、浴室、窗槽、踢脚线
- 沟通慢，改约麻烦

### 结果痛点
- 退租验房不过
- 房东 / 中介指出漏项
- 家里有孩子宠物，担心药剂刺激
- 付款、发票、报销、NDIS 流程不顺
