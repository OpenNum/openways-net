# 足病-Podiatry 行业资料包

本目录已按可直接交付/可继续扩写的方式整理，重点覆盖：
- 常见脚痛问题
- 糖尿病足
- 运动损伤
- orthotics（矫形鞋垫/足底矫形）
- referral / claim（转诊与支付）
- 合规边界
- 适合 AI 接入的流程

## 文件清单
- `00-行业研究.md`
- `01-本地样本.md`
- `02-内容表达与广告话术.md`
- `03-五文件映射建议.md`
- `足病-Podiatry-后续可追踪.md`
- `sources.md`

## 先看什么
1. 先看 `00-行业研究.md` 了解行业结构与机会。
2. 再看 `02-内容表达与广告话术.md`，用于获客内容与前台脚本。
3. 需要落地 AI 行动包时，直接用 `03-五文件映射建议.md`。
4. 持续追踪与迭代，用 `足病-Podiatry-后续可追踪.md`。

## 适用场景
- 当地足病诊所
- 面向糖尿病足、高频脚痛、跑步/运动人群的专科或综合门诊
- 需要做内容增长、前台转化、复诊管理、orthotics 流程优化、referral/claim 解释的团队

## 使用提醒
- 本资料用于业务研究、内容策划、流程设计，不替代临床判断。
- 广告与内容必须避免虚假保证、夸大疗效、误导性 claim。
- 涉及 Medicare / private health / workers comp / DVA / NDIS 等支付路径时，需按当地最新政策复核。
