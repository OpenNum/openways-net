# 03-五文件映射建议

> 目标：把“言语治疗 / Speech Pathology”行业资料映射成一个可落地的 AI 行动包，适用于机构内部运营、前台咨询、评估前后协作、内容生产与家属沟通。

## 整体评估

### 适合 AI 接入的程度
**高**。原因：
- 大量重复性沟通与文书
- 明显的儿童/成人双漏斗结构
- 评估前后有高频信息收集、解释、随访需求
- 学校/家属/GP/转诊网络需要不同版本表达

### 不适合 AI 直接替代的部分
- 诊断与正式临床判断
- 高风险吞咽安全决策
- 复杂神经或医学红旗判断
- 未经审核直接发送的个体医疗建议

### 推荐 AI 目标
1. 缩短前台咨询到评估预约的时间
2. 提高评估前资料完整度
3. 减少治疗师会后文书负担
4. 提升家属与学校执行率
5. 让机构形成一致、清晰、合规的表达体系

## 1. Quick Start Guide

### 目标用户
- 私立 speech pathology 机构负责人
- 前台/客服/运营
- 儿童或成人线治疗师
- 需要做内容和转化的市场人员

### 7 天内先上线的最小可用版本
1. 建立儿童咨询分诊表单
2. 建立成人咨询分诊表单
3. 建立评估后家长版总结模板
4. 建立学校/老师沟通模板
5. 建立 GP/转诊回信模板

### 优先顺序
- 第一优先：咨询分诊
- 第二优先：评估总结
- 第三优先：家庭练习/随访
- 第四优先：内容生产

### 成功标准
- 预约前资料完整率提升
- 咨询响应更快
- 治疗师写报告/总结时间下降
- 家长/家属“下一步不清楚”的投诉下降

## 2. Tool Guide

### 工具 A：智能预分诊表单
**用途**
- 区分儿童/成人
- 采集主诉、年龄、症状、转诊来源、学校/医生信息、红旗情况

**输入字段建议**
- 年龄段
- 主要担心问题
- 持续多久
- 是否已有诊断/病史
- 是否需要学校协作
- 是否存在吞咽/呛咳/突发恶化等红旗

**输出**
- 服务线归类
- 是否建议尽快人工回电
- 首次预约建议类型
- 需准备资料清单

### 工具 B：评估前资料助手
**用途**
- 自动提醒家长/家属准备病史、学校反馈、视频样本、出院小结等

### 工具 C：会后总结生成器
**用途**
- 把治疗师笔记转成家长版/家属版易读总结

**输出模板结构**
- 今天看到了什么
- 这意味着什么
- 近期优先目标
- 家里怎么配合
- 什么时候复查/复诊
- 哪些情况要尽快联系医生或机构

### 工具 D：学校/转诊协作文档器
**用途**
- 生成老师建议单、学校沟通摘要、GP 回信草稿、ENT/儿科协作摘要

### 工具 E：家庭练习与随访助手
**用途**
- 生成一周家庭练习
- 自动发送简短打卡问题
- 汇总家长反馈，供下次 session 使用

## 3. Prompt Library

### Prompt 1：儿童咨询分诊
“你是言语治疗机构的分诊助手。根据家长提供的信息，判断属于发音、语言、社交沟通、口吃、读写、喂养/吞咽中的哪一类或多类。输出：1）可能服务线，2）建议预约类型，3）需准备的资料，4）是否存在需要尽快人工跟进的红旗。不要做诊断，不承诺疗效。”

### Prompt 2：成人咨询分诊
“你是言语治疗机构的成人服务分诊助手。根据用户描述，区分可能属于神经沟通、声音、吞咽、认知沟通或其他方向。输出：1）建议服务线，2）就诊前需准备资料，3）是否建议同时联系 GP/专科，4）红旗提醒。不要给个体医疗结论。”

### Prompt 3：家长版评估总结
“根据以下临床笔记，生成一份家长能看懂的评估总结。结构包括：今天看到了什么、主要影响、优先目标、家里本周可做的 3 件事、是否建议与学校沟通。语气清晰、具体、不过度承诺。”

### Prompt 4：学校建议单
“根据儿童评估结果，写一份给老师的简洁建议单。内容包括课堂支持重点、沟通提示、任务调整建议、需要观察的表现。避免医疗化措辞，强调课堂可执行性。”

### Prompt 5：GP/转诊回信草稿
“根据本次评估与主诉，为 GP/转诊医生写一封专业但简洁的回信草稿，包含主诉、关键观察、功能影响、建议下一步。区分事实观察与建议，不做超范围结论。”

### Prompt 6：家庭练习单
“根据孩子/成人的本周目标，生成一份 7 天家庭练习单。每次 5-10 分钟，列出步骤、材料、示范语言、记录方式，并提醒家属如出现困难该如何反馈。”

## 4. Workflow SOP

### SOP A：儿童线
1. 家长提交咨询表
2. AI 归类问题类型，标记是否有红旗
3. 前台人工确认预约类型与资料清单
4. 评估完成后，AI 生成家长版总结 + 学校版建议草稿
5. 治疗师审核后发送
6. 一周后发送家庭练习跟进

**关键控制点**
- 任何疑似吞咽/明显退化/复杂神经问题必须人工复核
- 涉及学校沟通先确认家长授权

### SOP B：成人线
1. 患者或家属提交咨询信息
2. AI 归类为声音、神经沟通、吞咽或其他
3. 若涉及吞咽风险或突发症状，提示尽快人工处理/就医
4. 预约前自动索取转诊单、出院摘要、病史资料
5. 评估后生成家属版说明 + GP 回信草稿
6. 随访阶段生成居家执行与风险提醒

### SOP C：内容运营线
1. 收集高频咨询问题
2. AI 按儿童/成人分类
3. 输出短文、FAQ、短视频脚本
4. 合规审核后发布
5. 用内容反哺咨询分诊话术

## 5. 7-Day Plan

### Day 1
梳理当前服务线，明确儿童与成人分开命名、分开页面、分开咨询表。

### Day 2
建立儿童/成人咨询分诊字段与红旗规则。

### Day 3
建立评估后总结模板，至少包含家长版与家属版。

### Day 4
建立学校建议单、GP 回信、转诊协作模板。

### Day 5
建立家庭练习单与一周随访模板。

### Day 6
整理 20 个高频问题，生成 FAQ 与内容选题库。

### Day 7
试跑 3 个儿童案例、2 个成人案例，检查输出是否清晰、合规、可执行。
