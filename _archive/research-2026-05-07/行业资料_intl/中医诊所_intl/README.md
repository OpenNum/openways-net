# 中医诊所资料包

用途：为后续生成这个行业的页面、品牌、社媒内容提供前置研究与表达参考。

当前目录文件：
- `00-行业研究.md`
- `01-本地样本.md`
- `02-内容表达与广告话术.md`
- `03-五文件映射建议.md`

说明：
- 语境以当地 / 您所在的城市本地服务为主，重点面向当地华人本地服务市场。
- 本次重点聚焦“中医诊所”里最适合做本地获客页的主赛道，即 acupuncture / 针灸、Chinese herbal medicine / 中药、pain management / 疼痛管理、women’s health / fertility support、stress / sleep、digestion、injury rehab、以及 private health rebate / HICAPS / bilingual support 等高转化元素。
- 重点看的是：行业细分、目标客户、服务项目、获客渠道、痛点、卖点、本地样本、以及中英双语高频表达。
- 仅作品牌、页面、内容参考，不构成医疗建议，不替代医生面诊、诊断或紧急救助。
- 涉及疗效、疾病名称、孕产、 fertility、 chronic conditions 等内容时，必须遵守当地医疗广告规范，避免绝对化、保证疗效、误导性 therapeutic claims。
