# 02-内容表达与广告话术

## 1. 内容主轴
高压清洗最有效的表达，不是设备参数，而是“看得见的变化 + 风险可控 + 报价简单”。

核心顺序：
1. 脏污对比
2. 场景归类
3. 为什么现在该做
4. 报价门槛低，先发照片即可
5. 哪些情况需要现场复核

## 2. 高转化内容角度

### Driveway
- 你家门口最先被看到的地方，不该这么脏
- 油渍、轮胎印、青苔，几小时内明显改观
- 卖房前、出租前、雨季后，最适合做 driveway refresh

标题示例：
- 这条发黑 driveway，清完像换了地面
- 只洗 driveway，门面感就能回来一大截
- 油渍、青苔、轮胎印，driveway 最值得先做

### Roof
- 屋顶发黑发绿，不只是难看，也可能意味着藻类和苔藓累积
- Roof cleaning 重点不是“越猛越好”，而是安全处理、尽量不伤表层
- 屋顶项目先看照片，再决定是否上门复核

标题示例：
- Roof cleaning 不是一味高压冲，安全方法更重要
- 屋顶黑纹和青苔，不处理会越来越明显
- 先发照片，我们先判断是否适合做 soft wash

### Exterior / House Wash
- 外墙、檐口、围栏、门口一起做，整体感最明显
- 不一定大翻新，先把 exterior 洗干净，房子就顺眼很多
- 适合与 driveway 打包成“门面焕新套餐”

标题示例：
- 不装修，也能让房子看起来像刚整理过
- Exterior wash + driveway，最适合做整屋门面提升
- 出租/出售前，先把外墙和门口处理干净

### 商业
- 店门口的人行道和门头，就是第一印象
- 物业、商铺、园区更需要低打扰、可复用的维护计划
- 商业报价重点不是最低价，而是可安排、可交代、可持续

标题示例：
- Storefront 不干净，客流第一眼就扣分
- 商业外立面清洗，重点是低干扰和稳定维护
- 不是一次洗完就结束，而是建立季度维护节奏

## 3. Before-after 内容模板

### 图文模板
- 第 1 句：点出表面和问题
- 第 2 句：点出处理方式
- 第 3 句：点出结果
- 第 4 句：CTA，发照片报价

示例：
这条 driveway 长期堆积油渍、轮胎印和青苔，门口看起来一直灰灰黑黑。 这次先做表面预判，再针对污渍强度处理。 清完后，整体边界和颜色都明显回来。 如果你也想先知道自己家适不适合做，直接发 3 到 5 张照片就能先给区间报价。

### 短视频口播模板
- 开头 3 秒：最脏位置特写
- 中段：清洗过程 + 表面变化
- 结尾：完工全景 + 引导发图

口播示例：
这类 driveway 最常见的问题就是油渍、青苔和旧污层叠在一起。不是每种表面都适合同样压力，所以我们会先判断材质和脏污。像这种项目，先发照片，通常就能先判断大概难度和报价区间。

## 4. 广告话术

### 短广告文案
- Driveway 太脏先别急着重铺，先发照片看看值不值得洗。
- 屋顶发黑发绿，不一定适合直接高压冲，先看照片更稳。
- 外墙、门口、车道一起做，门面感提升最明显。
- 商铺门头、人行道、共用区域，可做非营业时段清洗。

### CTA
- 发照片，先给区间报价
- 先看图，再判断是否需要上门
- 免费初步评估
- 住宅可先做套餐价，商业可做维护报价

## 5. 客服/私信回复话术

### 照片报价引导
把 driveway / roof / exterior 的 3 到 5 张照片发来，再补一个大概面积和所在区域，我们可以先给你做初步判断。如果是屋顶或高风险项目，我们会先告诉你是否需要现场复核。

### Roof 风险边界
屋顶项目我们一般会先看照片判断材质、坡度和污染情况，不会默认建议强压处理。部分屋顶更适合 soft wash 或现场复核后再报价。

### 商业项目引导
商业项目麻烦补充一下位置类型、面积、营业时间、是否能安排非营业时段，以及是否需要长期维护。这样更容易给你一个可执行的方案，而不是只报一个单次价格。

## 6. AI 最适合直接生成的内容资产
- before-after 图文卡片
- 短视频口播脚本
- 私信回复模板
- 报价解释话术
- 商业维护提案开场文案
- 完工后索评与转介绍文案
