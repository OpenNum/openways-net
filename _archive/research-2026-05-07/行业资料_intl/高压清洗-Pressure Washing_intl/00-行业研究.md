# 00-行业研究

## 1. 行业一句话判断
高压清洗本质上是“低客单入门、靠细分场景拉开价差、靠内容和响应速度拿单、靠复购和组合服务放大利润”的本地服务行业。最容易标准化的是 driveway / path / exterior wash，最需要风险边界的是 roof cleaning。

## 2. 核心细分

### A. Driveway / concrete / path
- 最适合做 before-after 内容
- 污渍可视化强，广告转化直观
- 住宅端更容易照片初报
- 常见污渍：油渍、轮胎印、青苔、霉斑、泥渍
- 适合做套餐：driveway + path + front entry

### B. Roof cleaning
- 需求强，但风险最高
- 不宜简单默认“高压冲洗”，更常见安全表达是 low-pressure / soft wash
- 重点痛点：黑色条纹、苔藓、藻类、排水不畅、影响房屋观感
- 需要单独做免责声明：坡度、材质、脆化瓦片、旧屋、渗漏既有问题
- 更适合“照片预审 + 现场复核 + 风险确认”流程

### C. Exterior / house wash / facade
- 包含外墙、檐口、围栏、车库门、庭院硬质表面
- 住宅端常与 driveway 打包
- 商业端常变成 storefront / facade / signage / common area / bin area
- 可作为“维护感”服务，而不是一次性深度清洁

## 3. 住宅 vs 商业

### 住宅
决策逻辑：
- 更看重外观变化
- 更容易被 before-after、邻里对比、卖房前整理打动
- 更接受照片报价和套餐价
- 更容易追加 roof gutter exterior bundle

住宅常见触发场景：
- 房子准备出售或出租
- 季节性清洁
- 连续雨季后发霉发绿
- Driveway 太脏影响门面
- 房东 / 物业交房前整理

### 商业
决策逻辑：
- 更看重安全、形象、时段配合、保险、低扰民
- 报价往往需要面积、材质、营业时段、排水条件、是否化学剂限制
- 常见项目：店铺门头、人行道、停车区、垃圾房、共用通道、工业外立面
- 更适合月度 / 季度维护合约，而不是一次性单次清洗

商业成交关键：
- 可否非营业时段施工
- 是否 fully insured
- 是否能出风险评估或 SWMS 类文件
- 能否按门店 / 点位批量报价
- 能否提供前后对比记录，方便向物业或总部汇报

## 4. 市场共识与页面观察
基于当地与美国公开服务页面，本行业公开表达有几个稳定模式：
- 用“restore curb appeal / protect surfaces / remove mold mildew algae grime”描述价值
- 明确区分 pressure washing 与 soft washing
- Driveway、sidewalk、concrete 是高频展示项目
- Roof cleaning 会强调 soft wash、protect shingles / tiles、avoid damage
- 商业页会强调 insured、off-hours、property managers、retail storefront、HOA / property management body / mixed-use
- 本地品牌会强调免费报价、在线预约、保险、质保、快速响应

## 5. 风险边界

### 不要把所有项目都当成“高压清洗”
- 屋顶、老旧油漆面、木结构、脆弱勾缝、部分外墙材料不适合粗暴高压
- 错误水压会导致掉漆、进水、表层损伤、玻璃五金损坏

### Roof 特别风险
- 人身跌落风险
- 高压破坏瓦片、沥青瓦颗粒层、密封部位
- 旧屋可能本来就有裂缝或渗漏隐患
- 苔藓去除后可能暴露原有老化问题

### 化学剂与现场边界
- 药剂漂移可能影响植物、宠物、金属件
- 排水去向、雨水口、商业区公共区域要先确认
- 有些客户只想要“立刻变白”，但不接受药剂痕迹、等待时间或多次处理

### 报价边界
照片报价适合作为“初报/区间价”，不适合作为无条件最终价，尤其以下情况：
- 屋顶坡度大
- 高处作业
- 大面积油污或锈渍
- 商业面积大且需夜间施工
- 现场水电接入不明确
- 表面材质不清晰

## 6. 最适合 AI 接入的流程

### 流程 1，照片报价预审
输入：客户照片 + 地址/区域 + 要洗的位置 + 大致面积
AI 输出：
- 场景分类：driveway / roof / exterior / commercial
- 风险等级：低 / 中 / 高
- 价格区间
- 是否必须上门复核
- 需要补拍的照片清单

### 流程 2，线索自动分流
按字段自动归类：
- 住宅 / 商业
- 单次 / 周期维护
- 屋顶 / 外墙 / driveway / 综合包
- 紧急程度
- 是否适合短信成交

### 流程 3，before-after 内容生产
输入：施工前后照片、项目类型、污渍类型、区域
输出：
- Instagram/TikTok/Instagram/Facebook 文案
- 短视频口播脚本
- 广告标题与 CTA
- 客户案例卡片

### 流程 4，报价解释与异议处理
AI 自动生成：
- 为什么 roof 不直接高压冲
- 为什么照片只能先给区间价
- 为什么油渍/锈渍可能要单列附加项
- 为什么商业项目要按时段和面积报价

### 流程 5，完工后复购跟进
AI 自动触发：
- 7 天后索评
- 90 天后 driveway / exterior 复购提醒
- 雨季前 roof / gutter 检查提醒
- 商业客户季度维护建议

## 7. 经营抓手
- 获客抓手：before-after、邻里案例、卖房前焕新、照片快速报价
- 利润抓手：bundle 套餐、附加去污项、维护合约
- 交付抓手：标准化拍照、到场检查清单、完工记录
- 风险抓手：soft wash 判断、保险说明、现场边界确认、免责声明

## 8. 对 AI 项目的直接启示
如果后续要做 AI 行动包，优先级建议：
1. 照片报价分诊器
2. 住宅套餐报价助手
3. Roof 风险审单助手
4. before-after 内容生成器
5. 商业维护报价与跟进助手
