# 宠物服务 — 内容表达与广告话术

## 一、常见内容方向

### 1. 前后对比类
适合：宠物美容、洗护、造型

常见表达：
- 来之前像流浪小孩，回家像小王子 / 小公主
- 洗完之后整只都亮起来了
- 今日变身完成
- 从毛躁小狗，到精致宝宝
- 这次走的是韩系 / 清爽 / 软萌路线

### 2. 安心托付类
适合：寄养、日托、照看型服务

常见表达：
- 你上班 / 出门 / 旅行，我们帮你照顾好它
- 让毛孩子在安全、干净、有人陪伴的环境里待着
- 不只是托管，更是安心托付
- 每只宠物都有自己的节奏和照顾方式

### 3. 专业护理类
适合：高客单 grooming / spa / 特殊护理

常见表达：
- 根据宠物的毛发状态和性格安排护理方式
- 不是简单洗一洗，而是完整护理体验
- 兼顾清洁、舒适、造型和健康观察
- 每一次护理都更贴近宠物的个体需求

### 4. 节日 / 活动类
适合：促销、淡季拉新、旺季预约

常见表达：
- Easter / Mother's Day / Christmas / school holiday 特别活动
- 假期寄养名额开放
- 周末洗护时段开放预约
- 老客户回店福利
- 双宠 / 多次卡 / 组合项目优惠

### 5. 社区感 / 可爱互动类
适合：Instagram、TikTok、即时通讯工具社群

常见表达：
- 今日店里最会撒娇的是谁
- 今天又认识了新朋友
- 某只狗狗来店后的治愈瞬间
- 店里最受欢迎的小零食
- 本周 cutest face / funniest moment

## 二、常见客户沟通场景话术

### 1. 预约确认
- 您好，已经帮您预留 [日期] [时间] 的位置。
- 到店前如有特别护理需求，可以提前告诉我们。
- 如果宠物最近有皮肤、情绪或健康方面情况，也请提前说明。

### 2. 迟到提醒
- 为了不影响后面的预约安排，如果会迟到请尽早告诉我们。
- 如迟到超过 [X] 分钟，我们可能需要为您改约。
- 谢谢理解，我们也希望给每只宠物足够完整的服务时间。

### 3. 取消 / 改期说明
- 如需取消或改期，请尽量提前 [24/48] 小时通知。
- 临时取消可能会影响当天的预约安排。
- 我们会尽量帮您协调下一个合适时段。

### 4. 价格解释
- 不同价格主要取决于体型、毛量、毛发打结情况、所选造型和服务时长。
- 如果宠物需要额外护理或花费更多时间，我们会先和您说明。
- 我们希望收费和实际服务时间、难度相匹配。

### 5. 术后 / 护理后提醒
- 今天做完护理后，回家先让它休息一下。
- 如果有特别护理部位，我们会同步给您注意事项。
- 有任何不适应或问题，欢迎及时联系。

### 6. 寄养 / 日托说明
- 我们会根据宠物的性格、习惯和状态安排照顾方式。
- 如有饮食习惯、药物、分离焦虑、攻击性等情况，请务必提前说明。
- 首次寄养 / 日托建议先做基础沟通或试托。

### 7. 差评 / 客诉回复
- 谢谢您的反馈，很抱歉这次体验没有达到您的预期。
- 我们非常重视宠物的安全与舒适，也重视每位主人的感受。
- 如果您愿意，我们希望进一步了解情况，看看怎么补救和改进。

## 三、平台表达差异

### 1. 即时通讯工具 / 私聊
特点：
- 更直接
- 更适合确认信息、预约、解释价格和规则
- 适合一对一照顾感

关键词：
- 放心
- 细心
- 提前说明
- 帮您安排
- 我们会先确认

### 2. Instagram
特点：
- 图强于字
- 适合 before/after、可爱日常、轻品牌内容
- 英文/双语表达常见

关键词：
- fresh / clean / adorable / stylish / pampered
- happy pup / glow-up / grooming day

### 3. TikTok
特点：
- 视频节奏感强
- 适合 transformation、日托互动、搞笑片段、教育类短内容

关键词：
- before and after
- dog daycare routine
- what a grooming day looks like
- things pet owners should know

### 4. Google Review 回复
特点：
- 要简洁、专业、礼貌
- 不要太情绪化
- 好评回复强调感谢与期待再来
- 差评回复强调重视反馈与沟通意愿

## 四、常见广告角度

### 1. 安心型
- 你把时间留给工作和生活，我们把照顾留给毛孩子
- 不只是服务，更是把宠物交给值得信任的人

### 2. 精致型
- 不只是剪毛，是一次精致护理体验
- 让你的毛孩子干净、舒服、好看地回家

### 3. 专业型
- 根据体型、毛量、性格和状态做更合适的护理安排
- 每只宠物都不一样，护理方式也不该一样

### 4. 社区型
- 你家附近更懂宠物的门店
- 给狗狗交朋友，也给主人省点心

### 5. 节日促销型
- School holiday boarding spots now open
- 周末美容名额开放
- 节日限定造型 / 组合护理 / 老客回店福利
