# 支持性独立生活 Supported Independent Living

本目录整理了面向 Supported Independent Living, SIL 的行业资料，重点不是泛泛介绍，而是围绕可执行运营场景：入驻沟通、家属协同、住户匹配、支持人员交接、记录、投诉与边界管理，以及适合 AI 接入的流程。

## 文件结构
- `00-行业研究.md`：行业定义、监管逻辑、关键流程、国际对照、AI 机会点
- `01-本地样本.md`：当地本地及海外代表性样本
- `02-内容表达与广告话术.md`：官网、咨询、转化、家属沟通可直接复用的话术
- `03-五文件映射建议.md`：按 AI 行动包格式输出的五文件映射建议
- `支持性独立生活-Supported Independent Living-后续可追踪.md`：后续跟踪议题、信号与待补资料
- `sources.md`：本次整理参考来源

## 最关键结论
1. SIL 的核心不是“房子”，而是“持续且可审计的支持系统”。
2. 真正影响入住稳定率的不是获客，而是前期匹配、过渡准备、家属预期管理与班次交接质量。
3. 监管重点集中在 incident、complaints、records、consent、privacy、worker accountability，这些都非常适合 AI 做标准化与提醒。
4. 市场表达不能只讲“温暖照护”，还要讲“透明、可追踪、边界清晰、出问题有闭环”。
5. 最适合 AI 接入的环节不是替代 frontline support，而是 intake、匹配、handover、progress notes、family update、complaint triage、audit readiness。
