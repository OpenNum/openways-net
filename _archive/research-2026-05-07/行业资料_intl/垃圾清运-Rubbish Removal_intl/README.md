# 垃圾清运-Rubbish Removal

本目录为垃圾清运 / Junk Removal 行业资料重整版，面向本地生活服务、获客转化、客服分诊、报价、调度、以及 AI 接入设计。

## 目录
- `00-行业研究.md`：行业结构、商业模式、关键流程、AI 可切入点
- `01-本地样本.md`：当地、美国等代表市场样本与可借鉴做法
- `02-内容表达与广告话术.md`：落地页、广告、客服、短信、照片报价话术
- `03-五文件映射建议.md`：按 AI 行动包格式整理的知识文件映射建议
- `垃圾清运-Rubbish Removal-后续可追踪.md`：后续调研、竞品、数据更新建议
- `sources.md`：本次整理参考来源

## 本轮整理重点
1. 同日服务与紧急清运
2. 照片报价 / 远程预估
3. 体积计价、单件计价、到场 fixed quote
4. 禁运物与合规分诊
5. 客服前置筛查、到场窗口、短信通知、直播定位
6. 路线与调度优化
7. 适合 AI 接入的前台到后台链路

## 适合直接沉淀为 AI 的场景
- 客服首轮分诊
- WhatsApp / 短信 / 表单照片收集
- 体积预估与报价草案
- 禁运物拦截
- 同日单插单判断
- 司机排线与 ETA 通知
- 完工回访、评价、再营销
