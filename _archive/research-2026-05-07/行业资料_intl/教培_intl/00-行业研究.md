# 教培 / 语言培训 / 课外辅导行业研究

更新时间：2026-04-10

## 一句话判断
您所在的城市教培不是一个单一市场，而是至少由 4 条高频业务线组成：K12 补习、语言考试培训、大学/专业课辅导、社区语言与延展教育。华人市场里，用户真正买的不是“一节课”，而是提分、过关、升学、留学、移民、家长安心、全程中文沟通。

## 1. 行业细分

### A. K12 课外辅导 / 补习
典型场景：
- 小学英语、数学
- OC / Selective / NAPLAN 备考
- 初高中英文、数学、科学
- HSC / IB 等考试冲刺
- 一对一、小班、假期营、精英班

高频表达：
- K-12 tutoring
- HSC tutoring
- IB tutoring
- Selective / OC prep
- small group / 1-on-1
- tailored learning plan
- free trial lesson

### B. 语言培训
典型场景：
- ELICOS 英语学校
- IELTS / PTE / Cambridge / Academic English
- 日常英语 / 商务英语 / English for Work
- NAATI CCL、口笔译、移民英语

目标人群：
- 留学生
- 需要考语言成绩的移民 / 求职 / 申请学校人群
- 打工度假 / 海外访客

### C. 大学辅导 / 商科与专业课补习
典型场景：
- 会计、金融、经济、数据、IT、工程课程辅导
- 预科 / 本科 / 研究生 assignment、考试辅导
- CFA / CPA / FRM 等证书培训
- Excel / MYOB / Xero / 软件实操

### D. 社区语言学校 / 双语传承教育
典型场景：
- 周末中文学校
- 普通话 / 粤语课程
- heritage language 教育
- 文化活动 + 社区型家长参与

## 2. 主要目标客户

### 客户类型 1：华人家长
最关心：
- 提分与排名
- 老师是否靠谱
- 班型是否适合孩子
- 能不能长期跟得住
- 是否有真实成绩案例
- 是否能中文沟通

### 客户类型 2：留学生 / 国际学生
最关心：
- 能不能快速通过语言考试或课程考试
- 有没有短期冲刺
- 是否熟悉您所在的城市本地学校与课程体系
- 是否线上线下都可

### 客户类型 3：移民 / 求职导向成人
最关心：
- PTE / IELTS / NAATI 能否快速出分
- 课程是否有模考反馈
- 是否兼顾就业、签证、转学或职业路径建议

### 客户类型 4：重社区关系的家庭
最关心：
- 教育氛围是否稳定
- 孩子是否愿意去
- 学校是否正规、口碑好、办学久
- 是否有家长社群、文化认同

## 5. 用户痛点
- 不知道该选哪种课程线
- 害怕花钱没效果
- 担心老师水平不稳定
- 班型太大、孩子跟不上
- 信息过多，但缺少针对自己情况的建议
- 留学生 / 成人用户担心时间紧、目标分不清晰

## 6. 商家卖点
- 本地课程体系经验
- 成绩案例 / 通过案例
- 免费试听 / 免费测评
- 小班 / 一对一 / 分层教学
- 中英双语沟通
- 线上线下结合
- 阶段反馈与学习路径清晰

## 7. 页面与内容表达建议
- K12、语言考试、大学辅导不要混成一页主叙事
- 家长向内容强调：安心、进步、阶段反馈、长期陪伴
- 成人 / 留学生向内容强调：效率、提分、过关、路径清晰
- 尽量少写空泛“专业负责”，多写课程对象、适合谁、如何开始
