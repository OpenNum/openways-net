# 车辆美容-Car Detailing - 五文件映射建议

更新时间：2026-04-12

## 建议对应的 AI 行动包 5 文件

1. Quick Start Guide
- 哪类客户适合哪些美容项目
- 最容易成交的入口项目
- 典型客户场景

2. Tool Guide
- 报价与套餐收集
- FAQ 与客服辅助
- 预约说明模板
- before / after 内容整理

3. Prompt Library
- 项目介绍提示词
- 套餐解释提示词
- 镀膜教育提示词
- 售前整备沟通提示词

4. Workflow SOP
- 询盘到需求确认
- 车型和项目判断
- 预约与到场 / 上门安排
- 服务交付
- 售后与复购跟进

5. 7-Day Plan
- 第一天梳理服务入口
- 第二天整理 FAQ
- 第三天搭建报价 / 咨询表单
- 第四天写页面样本
- 第五天补 before / after 表达
- 第六天补售后脚本
- 第七天收口标准流程
