# 留学移民服务后续五文件映射建议

更新时间：2026-04-10

## 已理解的既有结构
参考当前 industry-packs 目录下已完成资料包逻辑，这类目录的作用是：
1. 先把行业研究做清楚
2. 再为后续页面 / 品牌 / 社媒生成提供统一依据
3. 让单个行业能独立收口，不和别的行业混在一起

## 我对“是否可映射到 5 文件”的判断
可以，而且很适合。

原因：
- 留学移民天然就是多阶段决策，不同阶段搜索意图完全不同
- 用户不是在搜一个抽象品牌，而是在搜“我现在这个阶段怎么办”
- 这个行业特别适合按阶段、签证类型、人生节点来拆页面

## 更推荐的 5 文件切法

### 方案 A，按“总入口 + 4 个核心阶段页”切
这是我最推荐的方式。

1. 首页 / 综合入口页
   - 讲城市、中文服务、团队、服务范围、预约方式
2. 留学申请页
   - 选校、选专业、申请、offer、COE、预算与时间线
3. 学生签证 / 续签页
   - student visa、续签、转学、补件、材料清单
4. 485 / 留学到移民路径规划页
   - 毕业后身份衔接、485、职业评估、州担保、PR 路线
5. 技术移民 / 配偶 / 复杂签证页
   - 可按品牌实际重心，在“技术移民 / 雇主担保”和“配偶 / 复杂案件”之间二选一

### 方案 B，按“高意图签证页”切
如果品牌偏移民，不偏教育中介，可用：
1. 首页
2. 学生签证页
3. 485 页
4. 技术移民 / 雇主担保页
5. 配偶 / 拒签页

### 方案 C，按资产类型切
如果“5 文件”指的是不同设计 / 内容资产：
1. 行业总包
2. website-design
3. pdf-design
4. poster-design
5. social-design

但就这个行业而言，我仍然更建议先走方案 A 或 B。

## 我更推荐哪种
我更推荐 **方案 A**。

因为真实流量入口往往是：
- 我该申请什么学校和专业
- 学生签证怎么办
- 我快毕业了 485 怎么接
- 我怎么从留学走到 PR
- 我的配偶签 / 拒签问题怎么办

也就是典型的“阶段驱动 + 问题驱动”。

## 还可以继续补的内容
如果决定继续深化，下一轮最值得补的是：
1. **更细的 FAQ 模版**
   - 学签 FAQ、485 FAQ、配偶签 FAQ、拒签 FAQ

   - Instagram/TikTok标题、朋友圈文案、视频号口播脚本
3. **更细的本地 suburb 语境**
   - your city CBD、your area、your area、your area 等
4. **更细的角色边界说明**
   - 教育顾问、注册移民代理、律师三者页面怎么写更合规

## 当前结论
- **可映射到 5 文件：是**
- **值得继续：是**
- **最值得优先继续的方向：学生签证、485 / 路径规划、技术移民 / 配偶签**
- **如果要快速落地获客页：先做综合首页 + 学签页 + 485 页 + 技术移民页**
