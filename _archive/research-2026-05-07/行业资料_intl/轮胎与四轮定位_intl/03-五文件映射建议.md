1. Quick Start Guide
- 哪类轮胎问题最常见
- 最容易成交的入口
- 典型客户场景

2. Tool Guide
- 车型与轮胎规格收集
- FAQ 与客服辅助
- 预约说明模板
- 价格说明模板

3. Prompt Library
- 服务介绍提示词
- 轮胎规格解释提示词

- 询盘到信息确认
- 车型 / 尺寸 / 症状判断
- 到店或预约安排
- 服务交付
- 售后跟进

5. 7-Day Plan
- 第一天梳理服务入口
